# WordPress Ollama AI 插件界面说明

## 项目简介

WordPress Ollama AI 插件是一个开源项目，旨在为 WordPress 网站提供 AI 聊天功能集成解决方案。

## 界面特点

- 简洁的聊天界面设计，支持实时对话交互
- 响应式布局，适配桌面端和移动端
- 支持 Markdown 格式文本显示
- 内置代码高亮显示功能
- 可自定义主题颜色和样式

## 主要功能区域

- 对话输入框：支持多行文本输入
- 消息展示区：清晰展示人机对话记录
- 设置面板：提供界面定制选项
- 系统提示：友好的操作引导和状态提示

## 技术实现

- 基于 React 构建的现代化前端界面
- 使用 WebSocket 实现实时通信
- 支持主题自定义的 CSS 变量系统
- 集成 WordPress REST API
